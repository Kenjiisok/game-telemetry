======================================
    COMO RECRIAR O EXECUTÁVEL
======================================

Se você precisar modificar o código e criar um novo .exe:

1. INSTALAR DEPENDÊNCIAS:
   pip install pyinstaller
   pip install -r requirements.txt

2. CRIAR EXECUTÁVEL:
   pyinstaller --onefile --windowed --name=KenjiOverlay --add-data="overlay.png;." tinypedal_inspired_overlay.py

3. RESULTADO:
   - Arquivo criado em: dist/KenjiOverlay.exe
   - Tamanho: ~67MB
   - Contém todas as dependências

4. DISTRIBUIR:
   - Apenas o arquivo KenjiOverlay.exe
   - + INSTRUÇÕES_PARA_AMIGOS.txt
